# Eight Bit Breadboard Computer

## Background

This 8 Bit computer is my build of the Ben Eater SAP-1 Computer. The entire build process is documented in a [YouTube course](https://www.youtube.com/playlist?list=PLowKtXNTBypGqImE405J2565dvjafglHU) and all the materials required to build the computer can be purchased from his website at [eater.net/shop](https://eater.net/shop).

I wanted to preface this project by saying that this computer was not my idea and to give credit where credit is due. Throughout the build process I did make several quality of life modifications and I will discuss those in detail, but the majority of the build looks and functions exactly like Ben Eater's original "8 Bit Breadboard Computer".
